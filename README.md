# Network Technologies Labs

This repository contains lab exercises for the **Network Technologies** course, focusing on network design, traffic analysis, and simulation tools.

## Overview

The labs cover fundamental concepts and practical skills in networking, including:

- Signal encoding and modulation methods  
- Fast Ethernet network calculations  
- Traffic analysis using Wireshark  
- Setting up experimental network environments with GNS3  
- Designing and analyzing simple networks in GNS3  

## Lab List

1. **Signal Encoding and Modulation Methods**  
2. **Fast Ethernet Network Calculations**  
3. **Traffic Analysis with Wireshark**  
4. **Setting up Experimental Network Testbed in GNS3**  
5. **Simple Networks in GNS3 and Traffic Analysis**  

## Usage

- Each lab includes instructions and exercises to practice network design and analysis.  
- Use Wireshark to capture and analyze network traffic.  
- Use GNS3 to simulate network topologies and test configurations.  

## Requirements

- Wireshark network analyzer  
- GNS3 network simulator  
- Basic understanding of networking concepts and protocols
